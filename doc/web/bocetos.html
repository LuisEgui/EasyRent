Página inicial

Esta será la página principal del portal web EasyRent. En ella se podrá acceder a las funcionalidades básicas, tales como:

  • Iniciar Sesión

    Para ello, el usuario debe pulsar el botón "Login".

  • Registrarse

    Para ello, el usuario debe pulsar en el botón desplegable "Menu" y posteriormente pulsar sobre el botón "Registrarse" situado en dicho desplegable.

  • Buscar vehículos disponibles

    Cualquier usuario será capaz de realizar búsquedas de vehículos indicando para ello: la ubicación de recogida del mismo, la fecha de recogida y devolución; y hacer "click" sobre el botón "Buscar".

  • Visualizar promociones publicadas hasta la fecha.
  • Obtener información legal, FAQ, de contacto.

    Dicha información se podrá acceder desde cada página del portal.

Menú (no registrado)

Este será el menú que se despliegue cuando el usuario no haya iniciado sesión previamente en el sistema. Tras hacer "click" sobre el mismo, podremos acceder a:

  • Registrarse
  • Foro
  • Contacto
  • FAQ

Menú (registrado)

Este será el menú que se despliegue cuando el usuario haya iniciado sesión previamente en el sistema. Tras hacer "click" sobre el mismo, el usuario podrá acceder a:

  • Reservas
  • Generar incidente
  • Bandeja de entrada
  • Tus promociones
  • Tu perfil
  • Foro
  • Cerrar sesión
